# Wild Wanderers — Design Reference

This folder is the visual source of truth for the build. The repo overrides
ambiguity, and within the repo, these files define the look.

- homepage-reference.html — the approved homepage mock. Open it in a browser.
  Every color, type choice, section order, and the ridgeline motif are here.
  The build should match this, then improve on motion and responsiveness.
- hero.png — the hero photograph (placeholder until Gabe sends real Baylands shots).
- fonts/ — Fraunces (display) and Plus Jakarta Sans (body), self-hosted.
- WildWanderers_Site_Spec.md — the full spec: tokens, type, motion, IA, build rings.

Do not restyle away from homepage-reference.html. Port it into a real
Next.js component architecture, keep the design identical, add the motion
described in the spec.
